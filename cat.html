Sinal de maior ">"

Esse sinal > é muito legal e divertido, pois ele faz coisas incriveis, com ele podemos transferir coisas para outro diretório/pasta, a função dele é direcionar Ex: "arq1 > file2" .
Ou seja eu direcionei o conteudo do "arq1" para o "file2". Temos o sinal >> diferente do sinal > ele adiciona algo para a pasta, digamos que você quer adicionar algo e não direcionar, usamos o seguinte comando Ex: arq2 >> file2
Aqui eu adicionei coisas do arq2 para a pasta file2.

Super usuário

Para você poder acessar o modo super usuário, você deve digitar o seguinte comando Ex: "su -" ele irá pedir sua senha de login, com ela você consegue adentrar no modo super usuário, tome cuidado pois com esse modo você tem acesso total a sua máquina, caso exclua algum arquivo ou programa sem querer, não tem como recuperar, pode acabar danificando sua máquina caso exclua algo que não deveria. Para você poder identificar se está no modo super usuário ou no usuário basico verifique por esses sinais no prompt Ex: ~$ = "usuário simples" ~# ="Super usuário".
Para poder sair do modo super usuário, apenas digite "exit".

Comando cat

Com o comando cat podemos fazer muitas coisas interessantes e legais, como por exemplo: Ver o que tem dentro de uma lista, Escrever em formato de lista, Numerar o que foi escrito em uma lista.
O comando cat serve para escrever algo na tela, exibir conteudo ou até mesmo juntar dois arquivos de texto. Vamos aos exemplos:
"cat > lista.txt << @" você começa a escrever em formato de lista, como se fosse um bloco de notas. com o CTRL+D você salva esse arquivo que foi criado
"cat -n lista.txt" faz a listagem do que foi escrito com numeros.
Para exibir o que está dentro dessa lista digite o comando "cat lista.txt" ele irá exibir na tela o que tem escrito.
O comando "cat lista.txt > lista2.txt" passa todo o conteudo do projeto.txt para o projeto2.txt, sobrescrevendo tudo por cima.
"cat lista.txt >> lista2.txt" Essa sintaxe concatena "junta" dois arquivos txt, colocando o conteudo na ultima linha que foi escrito, e não sobrescrevendo o conteudo.
Temos o comando de substituição de caracteres na tela com a seguinte sintaxe, "cat lista | a-z".
Você pode usar o comando man cat para visualizar o manual do cat, ou então usar o comando cat -help e você vera alguns comandos do cat.

Comando vi

O vi é um editor de texto padrão do linux que usamos no prompt de comando, ele é meio chatinha de se usar, porém simples, basta praticar. como iniciar, sair e comandos básicos para poder usá-lo.

Para iniciar use a sintaxe "vi nome_do_arquivo". Agora sempre que for fazer algum desses comandos abaixo, precione a tecla "Esq" mais a tecla que deseja Ex: Esq + i essa sintaxe é ultilizada para começar a escrever.

Comandos básicos:

  1. :wq - Salva o arquivo e sai do editor
  2. :w nome_do_arquivo - Salva o arquivo corrente com o nome especificado
  3. :w! nome_do_arquivo - Salva o arquivo corrente no arquivo especificado
  4. :q - Sai do editor
  5. :q! - Sai do editor sem salvar as alterações realizadas

Comandos de inserção:

  1. i - Insere texto antes do cursor
  2. a - Insere texto depois do cursor
  3. r - Insere texto no início da linha onde se encontra o cursor
  4. A - Insere texto no final da linha onde se encontra o cursor
  5. o - Adiciona linha abaixo da linha atual
  6. O - Adiciona linha acima da linha atual

Comandos de movimentação:

  1. Ctrl+f - Move o cursor para a próxima tela
  2. Ctrl+b - Move o cursor para a tela anterior
  3. H - Move o cursor para a primeira linha da tela
  4. M - Move o cursor para o meio da tela
  5. L - Move o cursor para a última linha da tela
  6. h - Move o cursor um caractere a esquerda
  7. j - Move o cursor para a próxima linha
  8. k - Move o cursor para linha anterior
  9. l - Move o cursor um caractere a direita
  10. w - Move o cursor para o início da próxima palavra (Ignora a pontuação)
  11. W - Move o cursor para o início da próxima palavra (Não ignora a pontuação)
  12. b - Move o cursor para o início da palavra anterior (Ignora a pontuação)
  13. B - Move o cursor para o início da palavra anterior (Não ignora a pontuação)
  14. 0 - Move o cursor para o início da linha atual
  15. ^ - Move o cursor para o primeiro caractere não branco da linha atual
  16. $ - Move o cursor para o final da linha atual
  17. nG - Move o cursor para a linha n
  18. G - Move o cursor para a última linha do arquivo

Comandos de busca:

  1. /palavra - Busca pela palavra ou caractere em todo o texto
  2. ?palavra - Move o cursor para a ocorrência anterior da palavra
  3. n - Repete o último comando / ou ?
  4. N - Repete o último comando / ou ? , na direção reversa
  5. Ctrl+g - Mostra o nome do arquivo, o número da linha corrente e o total de linhas

Comandos de substituição e deleção:

  1. x - Deleta o caractere que esta sob o cursor
  2. dw - Deleta a palavra, da posição atual do cursor ate o final
  3. dd - Deleta a linha atual
  4. D - Deleta a linha a partir da posição atual do cursor ate o final
  5. rx - Substitui o caractere sob o cursor pelo especificado em x(é opcional indicar o caractere)
  6. Rx - Substitui a palavra sob o cursor pela palavra indicada em x
  7. u - Desfaz a última modificação
  8. U - Desfaz todas as modificações feitas na linha atual
  9. J - Une a linha corrente a próxima
