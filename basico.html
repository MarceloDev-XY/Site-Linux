Comandos Básicos do linux

cd mkdir rmdir clear history pwd help ls -l grep find

Comando cd

O comando "cd" serve para navegação entre diretórios, para voltar ou acessar algum diretório EX: "cd projeto" projeto é o nome do diretório/pasta.
Para voltar um diretório você deve usar o comando "cd ..", note que o espaço entre cd e .. é importânte, pois o Linux é sensivel a espaços e diverge de letras Maiúsculas e Minúsculas. Caso você queira voltar para o diretório Raiz, use o comando "cd /".

Comando mkdir

Esse comando é bem interessante, ele cria diretórios/pasta, no Linux você raramente você irá clicar com o botão direito e clicar em criar pasta.
Usaremos o comando mkdir Ex: "mkdir projeto", pronto você criou uma pasta, note que "projeto" é o nome da pasta, no qual você pode nomear como desejar.

Comando rmdir

E se você precisar ou querer remover um diretório/pasta ? Como vimos acima o comando "mkdir" cria diretório/pasta, agora iremos aprender a como remove-lo.
Basta usar o comando "rmdir", esse comando remove diretórios/pasta Ex: rmdir projeto. E se eu digitar rmdir Projeto e não rmdir projeto ele irá remover ? Não, pois um está com o "P" Maiúsculo e não "p" minúsculo.

Comando clear

Esse comando serve para limpar o prompt, usamos esse comando para ficar mais visivel o que estamos escrevendo e para não ficar aquele monte de letras e códigos dificultando onde estão cada coisa.
Basta usarmos ele da seguinte forma clear, apenas digite clear e tudo será limpo Ex: clear.

Comando history

O comando "history" é bem legal também, pois como diz o nome, ele demonstra o histórico de comandos que você deu, isso é importânte para caso você precisar excluir algum arquivo ou programa que você baixou, basta ver no histórico.
Assim como o comando "clear" apenas digite "history" Ex: history.

Comando pwd

O "pwd" serve como um GPS, caso você esteja perdido, que não é dificil de acontecer, use o comando "pwd" ele irá informar em qual diretório/pasta você se localiza. Ex: pwd.

Comando help

Mais um comando interessante o comando "help", ele realmente é de grande ajuda, para usar o comando "help", apenas digite no prompt "help", e ele irá listar diversos comandos que você pode ultilizar no Linux, além de dar uma breve explicação dos comando. Ex: help

Comando ls -l

Esse comando é interessante, se no Windows nós usamos o comando dir para demonstrar as pastas e diretórios que temos, esse segue o mesmo conceito, Lista os as pastas e diretórios que você possui, um pouco mais divertido, pois ele deixa de uma cor diferente as pastas/diretórios.
Para usar ele faça da seguinte forma Ex: ls -l e ele irá listar para você.

Comando grep

Esse comando serve para identificação de caracteres em uma lista ou arquivo por exemplo: "grep -n ka lista" isso fará com que tudo que estiver com ka no começo, final, ou no meio da palavra seja identificado, com isso ficara uma marcação.
Você ultilizara o comando grep + ação desejada.

Comandos grep:

  1. grep -n -lista todos que que forem identificados com determinada letra
  2. grep lu ou grep -i LU - lista tudo
  3. grep -in LU - lista por linhas

Extras

O comando find serve como um localizador de arquivos, a sintaxe dele é a seguinte exemplo: "find / -name".txt"" com esse comando você é capaz de identificar qualquer arquivo com a extensão.txt.

Temos o comando tail e head, que pega de uma posição a outra, eles funcionam da seguinte forma, eu quero identificar determinas linhas de uma lista, usarei a seguinte sintaxe Ex: tail-"13"lista head-"20" lista . Eu irei pegar da linha 13 até a 20, serão exibidos para mim as linhas 13,14,15,16,17,18,19 e 20.
